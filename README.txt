在线商城项目。
1.使用IDEA开发工具，结合使用Maven，将项目划分为DAO, Service, Control等子项目模块进行开发
2.使用Duild作为数据库连接池
3.使用Mysql作为数据库
4.使用Tomcat作为后端服务器
5.采用Spring+SpringMVC+Mybatis框架，实现了用户登录，商品分页浏览，商品搜索，管理员登录验证，商品上架，商品分类，商品管理，系统用户管理
6.使用聚合数据实现了短信快捷登录
7.使用Ajax实现了页面与web服务器的异步传输
8.结合使用pageHelper和bootstrap paginator框架实现了分页功能
